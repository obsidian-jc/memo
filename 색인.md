# 🗂 시설일지 색인

이 페이지는 손으로 정리하지 않아도, `시설일지`와 `메모` 폴더에 쌓이는 파일들을 자동으로 모아서 보여줍니다. 파일이 새로 생기거나 바뀌면 이 페이지를 다시 열 때마다 알아서 최신 상태로 갱신됩니다.

---

## ⚠️ 지금 처리 안 된 것 (이상 발견)

```dataview
TABLE date as "날짜", location as "장소", person as "담당자"
FROM "시설일지"
WHERE status = "×"
SORT date DESC
```

## 🔶 주의 상태

```dataview
TABLE date as "날짜", location as "장소", person as "담당자"
FROM "시설일지"
WHERE status = "△"
SORT date DESC
```

---

## 📅 최근 시설일지 (전체, 최신순 20개)

```dataview
TABLE date as "날짜", location as "장소", status as "상태", person as "담당자"
FROM "시설일지"
SORT date DESC
LIMIT 20
```

---

## 📍 장소별로 모아보기

```dataview
LIST
FROM "시설일지"
GROUP BY location
SORT rows.date DESC
```

---

## 📞 업체연락처 (분야별)

```dataview
TABLE title as "업체명", category as "분류", phone as "전화번호"
FROM "메모/업체연락처"
SORT category ASC, title ASC
```

---

## 📝 메모 전체 (분류별)

```dataview
TABLE category as "분류", date as "날짜"
FROM "메모"
SORT category ASC, date DESC
```

---

## 📥 받은편지함 (일일 기록 원본)

```dataview
LIST
FROM "받은편지함"
WHERE type = "일일기록"
SORT date DESC
LIMIT 14
```

---

### 사용 팁
- 표의 **날짜/장소 컬럼 머리글을 클릭**하면 정렬 기준이 바뀝니다.
- 특정 장소만 보고 싶으면 `WHERE location = "지하2층 열교환실"` 처럼 조건을 바꿔서 별도 쿼리를 추가할 수 있습니다.
- 이 페이지 자체를 수정해도 원본 기록 파일에는 전혀 영향이 없습니다 (색인은 읽기 전용 조회일 뿐입니다).
